
# Employee Management Dashboard

## Overview
A React.js based employee management dashboard built using Material-UI.
Includes authentication, CRUD operations, search, filtering, dashboard analytics and theme switching.

## Tech Stack
- React.js
- Material-UI (MUI)
- React Router
- LocalStorage (Mock API)

## Features
- Login / Logout (Mock Authentication)
- Employee CRUD operations
- Search & Multi-filter (Gender, Status, State)
- Dashboard summary cards
- Graphical overview (Active vs Inactive)
- Light / Dark mode
- Confirmation dialogs
- Responsive UI

## How to Run Locally
1. Clone the repository
2. npm install
3. npm run dev
4. Open http://localhost:5173

## Assumptions
- Authentication is mocked
- Data persistence via localStorage
